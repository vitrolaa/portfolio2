NetGuardian

Sistema completo de monitoramento de dispositivos de rede, oferecendo uma solução simples, segura e intuitiva para verificar conectividade, armazenar logs e gerenciar um inventário de dispositivos em um único lugar. Essencial para ambientes onde a estabilidade de rede é crítica.

Tecnologias Utilizadas

Node.js Express MySQL PostgreSQL SQLite REST API

Funcionalidades Principais

  • Cadastro completo de dispositivos (nome, IP, localização)
  • Teste de conectividade via Ping em tempo real
  • Armazenamento e consulta de logs históricos
  • Inventário organizado de todos os dispositivos
  • Interface web simples e intuitiva
